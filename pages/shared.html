use class layout as a wrapper for a page or a container on a page

use variables from variables.css instead of hardcoding colors and fonts
this makes styles consistent and will make a light theme possible later
examples:

color: var(--text-color);
background: var(--background-color);
border-color: var(--color-dark-gray);

font-family: var(--font-text); // Default font
font-family: var(--font-title); // Title font
font-family: var(--font-button); // Button font

etc.
